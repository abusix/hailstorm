# Typography

The typography system uses two font families and a set of predefined text styles.

## Font Families

| Token          | Font             | Usage                             |
| -------------- | ---------------- | --------------------------------- |
| `font-display` | Fabriga          | Headlines, titles, prominent text |
| `font-sans`    | Inter            | Body text, UI elements            |
| `font-mono`    | System monospace | Code, technical data              |

```tsx
<h1 className="font-display">Display heading</h1>
<p className="font-sans">Body text</p>
<code className="font-mono">Code snippet</code>
```

## Usage Guidelines

### Headlines

- Use `headline-900` for page titles
- Use `headline-700` or `headline-600` for section headings
- Use `headline-400` or `headline-300` for card titles and labels

### Paragraphs

- Use `paragraph-300` for main body content
- Use `paragraph-200` for secondary content and descriptions
- Use `paragraph-100` for captions, helper text, and metadata

### Best Practices

1. **Maintain hierarchy** - Use consistent headline sizes to create clear visual hierarchy
2. **Pair with semantic text colors** - Combine typography classes with text color tokens
3. **Consider context** - Use smaller sizes for dense UI, larger for marketing pages

```tsx
// Page title
<h1 className="headline-900 text-text-tertiary">Dashboard</h1>

// Section heading
<h2 className="headline-600 text-text-primary">Recent Activity</h2>

// Card title
<h3 className="headline-400 text-text-primary">User Settings</h3>

// Body text
<p className="paragraph-200 text-text-secondary">
  Configure your account preferences and notification settings.
</p>

// Caption
<span className="paragraph-100 text-text-secondary">
  Last updated 5 minutes ago
</span>
```
