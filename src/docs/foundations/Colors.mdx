# Color Tokens

The color system is built on two layers: **primitive colors** and **semantic tokens**.

## Primitive Colors

Primitive colors are the raw color values. They should rarely be used directly in components.

## Usage Examples

```tsx
// Background tokens
<div className="bg-bg-primary">Primary surface</div>
<div className="bg-bg-secondary">Secondary surface</div>
<div className="bg-bg-danger-subtle">Danger alert background</div>

// Text tokens
<p className="text-text-primary">Primary text</p>
<p className="text-text-secondary">Secondary text</p>
<span className="text-text-danger">Error message</span>

// Border tokens
<div className="border border-border-primary">Bordered element</div>
<input className="border-border-secondary focus:border-border-brand" />

// Icon tokens
<Icon className="fill-icon-primary" />
<Icon className="fill-icon-success" />

// State combinations
<button className="bg-bg-brand hover:bg-bg-brand-hover active:bg-bg-brand-pressed">
  Brand button
</button>
```
